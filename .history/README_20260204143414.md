## 功能特性 (Features)

- **智能选区触发**: 鼠标抬起时自动检测选区并显示浮窗（仅在有效选区时显示）。
- **即时操作**:
  - 🔗 **打开链接**: 智能识别 URL，非标准链接自动转为搜索。
  - 🔍 **搜索**: 使用默认浏览器搜索选中文本。
  - 📋 **复制**: 一键复制到剪贴板（支持剪贴板历史恢复，避免非复制操作污染）。
  - 🌐 **Bob 翻译**: 深度集成 Bob 翻译软件，支持多版本（Standard/Setapp/Legacy）自动检测。
  - ⚡️ **LocalRAG Hub**: 集成 LocalRAG Hub 快捷键（`Cmd+K` 模拟）唤起。
- **系统集成**:
  - 🖥 **状态栏图标**: 快速访问设置、重启或退出应用。
  - 🚀 **开机启动**: 支持登录时自动启动（Launch Agent 方案）。
  - ⚙️ **设置面板**: 可自定义触发规则、外观大小、颜色模式等。
- **隐私安全**:
  - 🔒 **无网络请求**: 仅本地运行（除用户主动触发的搜索/链接外）。
  - 🛡 **权限管理**: 按需申请辅助功能和输入监控权限。

## 安装与运行 (Installation)

### 1. 下载应用

请前往 [Releases](https://github.com/henery/ClipLike/releases) 页面下载最新版本 `ClipLike.app`。

### 2. 首次运行

1. 将 `ClipLike.app` 拖入 `/Applications` 文件夹。
2. 双击打开应用。
3. 按照提示授予 **辅助功能 (Accessibility)** 和 **输入监控 (Input Monitoring)** 权限（这是监听选区和鼠标事件所必需的）。

### 3. 开发构建

如果你想自己构建项目：

**环境要求**:

- macOS 13.0+
- Xcode 14.0+

**构建步骤**:

```bash
# 克隆项目
git clone https://github.com/henery/ClipLike.git
cd ClipLike

# 使用一键打包脚本（推荐）
./build_app.sh
```

或者使用 Xcode 打开 `ClipLike.xcodeproj` 直接运行。

## 使用说明 (Usage)

1. **基本使用**: 在任意 App（浏览器、编辑器、微信等）中用鼠标选中文本，ClipLike 浮窗会自动出现。
2. **设置**: 点击菜单栏的 ClipLike 图标，选择“设置...”进入偏好设置。
3. **Bob 翻译**: 确保你已安装 [Bob](https://bobtranslate.com/) 翻译软件。ClipLike 会自动寻找并调用 Bob 进行翻译。
